Why?

My name is Dan Lasker and I created Calculator Croc to improve upon the current calculators you find when doing Google searches. Many of these sites are old, boring, cluttered, and lack information/features. My goal is to provide a more modern, user-friendly calculator site using the lastest and best web technologies and practices to allow you to perform the calculations you want and get the information you need.

How?

To create Calculator Croc I used the lastest web technologies and practices. I used a responsive design making my website work on both desktops and mobile devices. I also heavily used jQuery, jQuery UI, CSS3, and HTML5 to make my website beautiful, intuitive, and resposive to user input. I also use URL re-writing, AJAX, and the HTML5 history API to allow my site to function like a single page application(plus the back button still works!) but also have the capability to work as individual pages if the user's browser is out of date. Finally, I used AngularJS, Google's MVW javascript framework, to allow my calculators and content to update in real time as the user changes input and makes selections. All these technologies and design techniques make Calculator Croc the best and most modern calculator site on the web.
